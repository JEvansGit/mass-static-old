M.A.S.S Support

Questions and Answers

Q1: What is M.A.S.S ?
M.A.S.S is an acronym which stands for Machine-Assisted Surveillance System. This is a project which is aimed at creating a surveillance system using Responsive Web Design.
Q2: Why use a responsive Website for this ?
Responsive Web Design allows developers to easily create designs that are capable of adapting to multiple resolution through the use of Media queries.
Q3: Why doesn't this work?
At the moment, M.A.S.S is a project to demonstrate the use of responsive web design in the implementation of surveillance system, thus functionality such as a face-recognition is not required.
