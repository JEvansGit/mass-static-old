M.A.S.S

M.A.S.S (or Machine-Assisted Surveillance System) is a Surveillance system that incorporates Machine Learning to recognise faces that is easy to use and deploy.

The goal of this project is to create a responsive web-site that will be easy-to-use across multiple devices, including Desktops, Mobiles, and Smart Watches.
